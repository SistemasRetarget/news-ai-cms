### Configurar OAuth

```
1. Ir a Google Cloud Console
2. APIs & Services > Credentials
3. Crear OAuth 2.0 Client ID
4. Agregar redirect URIs:
   ├─ https://retarget-mcp-XXXXX.run.app/auth/google/callback
   └─ https://api.retarget.cl/auth/google/callback (si tienes dominio)
5. Guardar Client ID y Secret en Secret Manager
```

### Testear Conexiones

```bash
# Test endpoint context
curl -X GET \
  "https://retarget-mcp-XXXXX.run.app/api/v1/context/1" \
  -H "Content-Type: application/json" \
  -H "X-AI-Tool: claude"

# Test endpoint feedback
curl -X POST \
  "https://retarget-mcp-XXXXX.run.app/api/v1/feedback" \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": 1,
    "type": "suggestion",
    "content": "Feedback de prueba",
    "ai_tool": "claude"
  }'

# Test reportes

  "https://retarget-mcp-XXXXX.run.app/api/v1/reports/adoption"
```
