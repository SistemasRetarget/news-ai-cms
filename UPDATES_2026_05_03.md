## 1. Theme System + Color Selector

### Archivos Creados

- **`src/lib/themes.ts`** — Sistema de temas con 4 temas built-in
- **`src/lib/useTheme.ts`** — Hook de React para manejo de temas
- **`src/components/admin/ThemeSelector.tsx`** — Componente selector visual
- **`src/components/admin/ThemeSelector.module.css`** — Estilos del selector
- **`docs/THEME_SYSTEM.md`** — Documentación completa

### Features

✅ **4 Temas Built-in:**
- Light (Default)
- Dark
- Light (High Contrast)
- Dark (High Contrast)

✅ **Contraste WCAG Automático:**
- Cálculo real de ratios de contraste
- Badges en el selector (AAA, AA, Low)
- Validación en runtime

✅ **Persistencia:**
- localStorage con clave `cms-theme-preference`
- Sincroniza automáticamente entre tabs
- Se restaura al recargar

✅ **Variables CSS Dinámicas:**
- 11 variables de color disponibles
- Usables en cualquier componente
- Hot-switching sin reload

### CSS Variables Disponibles

```
--color-primary          (Brand color)
--color-secondary        
--color-accent          
--color-background      
--color-surface         
--color-text            
--color-text-secondary  
--color-border          
--color-success         
--color-warning         
--color-error           
```

### Cómo Usar

```tsx
// En componentes React
import { useTheme } from '@/lib/useTheme';

const { theme, selectTheme } = useTheme();

// En CSS
button { background: var(--color-primary); }

// En Dashboard Admin
<ThemeSelector />
```
